# Sokol International Circuit — Presentation

Speaker-led pitch deck for **Sokol International Circuit** (autodrom near Almaty, Kazakhstan — [sokol.kz](https://sokol.kz)), built to show an FIA delegation that the circuit is ready to host a world-class event.

The deck is a single, zero-dependency HTML file generated by a Python script.

## Structure

| Path | What it is |
| --- | --- |
| `_build_deck.py` | The deck generator. Injects SVG maps, the events tachometer, logos and styles into the HTML. |
| `sokol-deck.html` | The built presentation (19 slides, fixed 1920×1080 stage). Open in any browser. |
| `assets/` | Photos, logos (`assets/logos/`), the satellite-zoom frames (`assets/zoom/`), cutouts, and `logo.svg`. |
| `components/` | Standalone HTML widgets and the Python builders for the track map / roads. |
| `ads/` | The separate LED-ads sales deck pipeline. |
| `PLAN.md` | Content plan and slide outline. |

## Build

```bash
python _build_deck.py        # writes sokol-deck.html
```

Then open `sokol-deck.html` in a browser.
**Keys:** `←` `→` / `Space` to navigate · `E` for inline edit mode · `Ctrl+S` to save an edited copy.

## Notes

- Large/generated artifacts are git-ignored: rendered video (`*.mp4`), PowerPoint exports (`*.pptx`), the `export/` frame sequences, and screenshot folders. Regenerate them from the source scripts as needed.
- An editable, native version of the deck also lives in Figma (FIA Central Asia Drifting Cup 2026 file).
